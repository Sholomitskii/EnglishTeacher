    Документация к телеграм-боту EnglishTeacher.

    1.Данный телеграм-бот предназначен для изучения слов английского языка.
    2.Для начала использования телеграм-бота введите команду /start. После этого введите команду /cards чтобы начать
изучать новые слова.
    3.Телеграм-бот предлагает слово на русском языке, перевод которого нужно угадать, и 4 варинта ответа на 
английском языке. В случае верного ответа бот выдаёт сообщение "Верно!" и предлагает угадать новое слово.В случае
неверного ответа - сообщение "Попробуй ещё раз, у тебя получится!". Количество попыток для отгадывания не ограничено. 
    4.Для того, чтобы пропустить слово, нажмите кнопку "Дальше ->".
    5.В телеграм-боте доступна функция добавления и удаления слов из словарной базы. Для этого используйте кнопки 
"Добавить слово" и "Удалить слово" соответственно.

    Приятного использованию!
